# 🚀 Loaner Request Management Application

## 📌 Project Overview

The Loaner Request Management Application is a custom ServiceNow application developed using App Engine Studio to streamline the process of requesting and managing loaner devices within an organization.

The application enables employees to request devices such as laptops, mobiles, tablets, and monitors through a centralized Service Catalog while providing request tracking, approval management, reporting, and dashboard analytics.

---

### 👨‍💻 My Learning Journey

This project was developed as part of my ServiceNow learning journey to gain hands-on experience with App Engine Studio, Service Catalog, Flow Designer, Reports, and Dashboards.

## 🎯 Business Problem

Organizations often manage loaner devices manually through emails and spreadsheets, making it difficult to track requests, approvals, and device allocation.

This application provides a centralized and automated solution for managing loaner device requests efficiently.
