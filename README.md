# servicenow-loaner-request-management
Loaner Request Management Application built using ServiceNow App Engine Studio, Service Catalog, Flow Designer, Reports, and Dashboards.
Loaner Request Management Application
Project Overview

This project is a custom ServiceNow application developed using App Engine Studio to manage and track loaner device requests within an organization.

The application allows employees to request devices such as laptops, tablets, mobiles, and monitors through a centralized Service Catalog while enabling approval tracking, reporting, and dashboard analytics.

# Features

Custom ServiceNow Application
Loaner Request Table
Service Catalog Item
Catalog Variables
Approval Workflow
Request Tracking
Record Management
Reports
Dashboards

# Technologies Used
ServiceNow App Engine Studio
Flow Designer
Service Catalog
Forms & Tables
Reports & Dashboards

# Workflow

Employee Request
→ Service Catalog
→ Loaner Request Record
→ Approval Process
→ Status Update
→ Reports & Dashboard
